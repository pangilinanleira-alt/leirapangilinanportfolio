  • Lesson 1: Typography CSS

  • Typography: Styling and arrangement of text using CSS for legibility and visual appeal.

    Key Properties:

    font-family: Specifies typeface priority.

    font-size: Sets text size (e.g., px, pt, %).

    font-weight: Defines text boldness (100-900).

    font-style: Controls text style (normal, italic).

    line-height: Adjusts vertical space between lines.

    text-align: Specifies horizontal alignment (left, right, center, justify).

    color: Sets text color.

    text-decoration: Adds decorations (underline, line-through).

    text-transform: Changes text case.

  • Lesson 2: Formatting Text with CSS

  • Key Properties:

    Bold and Italic Text: Use CSS properties for styling.

    Indenting Text: Utilize for paragraph indentation.

    Changing Text Case: Apply for case adjustments.

    Line Spacing: Modify spacing with .

    Text Background Color: Use property for text color without altering page background.

  • Lesson 3: Formatting Layout with CSS

  • Key Properties:

    Box Model: Elements are structured in boxes with margins, borders, padding, and content.

    Positioning: Four types - static, relative, absolute, fixed; controls element placement.

    Offset: Sets distance from other elements; works with position property.

    Float: Controls block element placement; requires width specification.

    Background Properties: Control image placement, repetition, and attachment.

    Margins: Set using margin property for spacing around elements. Text Background Color: Use property for text color without altering page background.

  • Lesson 4: Formatting Border & Table

  • Key Properties:

    Borders: Customize borders using border-left, border-right-width and border-top-style property, and other properties.

    Padding: Defines space between cell content and border; can be set individually or uniformly.

    Table Background: Use property to set table background color.

    Table Style: Apply styles for highlights, colors, and text alignment in table cells.

    Table Layout: Utilize element for web page layout.
